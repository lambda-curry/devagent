# Generate Revise Report

## Mission
Generate a comprehensive revise report for a Beads Epic, aggregating traceability data and revision learnings from child tasks.

## Inputs
- Required: Epic ID (e.g., `bd-1a2b`)
- Optional: Output directory (default: `.devagent/workspace/reviews/`)

**Usage:** `devagent ralph-revise-report <EpicID>`
